# Authentication-App
An authentication app based on json web token. Refresh rotation approach has been used to build the project.

## Features
### 1) Register
### 2) Verify Email
### 3) Re-Send Verify Email 
### 4) Login
### 5) Refresh Token Rotation Approach
### 6) Logout
### 7) Forget Password
### 8) Reset Password

## Built With
### 1) Nextjs
### 2) Shadcn
### 3) Redux-Toolkit
### 5) Axios
### 6) React HooK Form
### 7) Expressjs
### 8) Postgresql Using Typeorm orm
### 9) Nodemailer
